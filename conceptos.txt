HttpOnly: El atributo HttpOnly es una bandera de seguridad de cookies que impide que los scripts del lado del cliente
accedan a las cookies. Esto se hace para evitar vulnerabilidades de scripts entre sitios, como el robo de cookies por 
medio de scripts malintencionados. 
Cuando se configura el atributo HttpOnly, los navegadores web solo envían las cookies en respuesta a solicitudes HTTP. 
Esto significa que los métodos que no son HTTP, como las referencias de document.cookie JavaScript, no pueden acceder a
la cookie. 
Para comprobar si una cookie de sesión tiene el indicador HttpOnly, se puede comparar el dominio con 
https://securityheaders.com. También se puede validar con las herramientas para desarrolladores de Google Chrome al 
examinar el encabezado de respuesta HTTP Set-Cookie. 
Aunque el atributo HttpOnly protege la confidencialidad de las cookies, no las protege de ser sobrescritas. Esto se 
debe a que un navegador solo puede almacenar un número limitado de cookies para un dominio.
